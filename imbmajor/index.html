FEATURES

IMB has following features to grab your attention.

  • Email Classification Based on Supervised Learning.

  • Build in JAVA with embedded database Derby for storing mails for the client.

  • Classification Based on Naive Bayes Classifier.

  • Stemming done with the help of Potters Algorithm

  • Easy to Install.

  • Create as many classifiers you want.
